# Assignment 2 - Android App

## Description
This is a simple Android app with two activities:
- **Main Activity:** Displays my full name and student ID. It has two buttons:
  - Start Activity Explicitly
  - Start Activity Implicitly
- **Second Activity:** Displays five mobile software engineering challenges and a button to return to the Main Activity.

## Tools Used
- **IDE:** Android Studio
- **Language:** Java
- **Emulator:** Pixel 7 (Android 14)

## Features
- Explicit and Implicit Intents
- Two Activities
- Button navigation between activities
